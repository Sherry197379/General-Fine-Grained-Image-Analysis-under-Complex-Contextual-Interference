The Google Cars dataset is specifically designed to evaluate fine-grained vehicle recognition under complex contextual interference. Due to its large size, it is hosted on Google Drive.

1. Download Link
Google Drive:https://drive.google.com/file/d/1cTL7GGQkUPRufbd-XRabKnb8MWH0HkbO/view?usp=drive_link

2. Dataset Structure
After downloading, please extract the contents into the data/ directory of this project. The expected structure should be as follows:
.
├── data/
│   └── GoogleCars/
│       ├── train/              t
│       ├── val/
